# Black-hole splash source

`assets/blackhole.gif` is the blackhole preview GIF from the `milli.nvim` media branch:

```text
https://raw.githubusercontent.com/amansingh-afk/milli.nvim/media/previews/blackhole.gif
```

Blak uses this asset only to generate a compact ASCII frame table at:

```text
lua/blak/splash/frames/blackhole.lua
```

The milli.nvim runtime is not vendored.
