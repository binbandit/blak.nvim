BLAK EXTRAS                                             *blak-extras*

Extras are optional modules. They can add Treesitter parsers, LSP servers,
Mason tools, formatters, linters, keymaps, Snacks modules, or plugin specs.

Commands: >
  :BlakExtras list
  :BlackExtras list
  :BlakExtras enable lang.typescript
  :BlakExtras disable lang.typescript
  :BlakExtras sync
<

State is stored outside the repository in:
  stdpath('state')/blak/extras.json

BlackExtras is an alias for BlakExtras.

Enabling an extra applies its config to the current session. Run :BlakExtras
sync or :Lazy sync if the enabled set added or removed plugin specs. Disabling
persists immediately, but restart Blak to unload plugins, keymaps, and runtime
hooks that already ran.
